# Política de Segurança

## Versões Suportadas

Acompanhamos o branch `main`.

## Relatando uma Vulnerabilidade

**NÃO** abra uma Issue pública para explorações de segurança.

Se você encontrar uma vulnerabilidade de segurança (ex: uma skill que ignora a verificação de "Uso Autorizado Apenas" ou executa código malicioso sem aviso):

1. E-mail: `security@antigravity.dev` (Espaço reservado)
2. Ou abra um **Aviso Privado (Private Advisory)** neste repositório.

## Política de Skills Ofensivas

Por favor, leia nossas [Salvaguardas de Segurança](docs/SECURITY_GUARDRAILS.md).
Todas as skills ofensivas são estritamente para **uso educacional e profissional autorizado apenas**.
